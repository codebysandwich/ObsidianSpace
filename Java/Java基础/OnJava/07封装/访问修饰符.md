Java中一共4总访问修饰符，`public`、 `protected`、`default`、`private`。对应的访问权限如下：

| 修饰符    | 当前类 | 同包 | 子类 | 其他包 |
| --------- | ------ | ---- | ---- | ------ |
| private   | ✅     | ❌   | ❌   | ❌     |
| default   | ✅     | ✅   | ❌   | ❌     |
| protected | ✅     | ✅   | ✅   | ❌     |
| public    | ✅     | ✅   | ✅   | ✅     |

> 其中子类指的是外部包中的子类
